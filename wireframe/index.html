Wireframe PPSB (Penerimaan Peserta Santri Baru)

Wireframe lengkap untuk sistem pendaftaran santri baru. Pilih halaman yang ingin dilihat.

Halaman Publik

  • Homepage
    Halaman utama untuk pengunjung website
  • Formulir Pendaftaran
    Form pendaftaran santri baru (3 tahap)
  • Konfirmasi Pendaftaran
    Halaman setelah berhasil mendaftar
  • Cek Status Pendaftaran
    Halaman untuk login dan mengecek status pendaftaran
  • Pembayaran
    Halaman untuk upload bukti pembayaran (pendaftaran & daftar ulang)

Halaman Santri (User)

  • Dashboard Santri
    Halaman utama setelah santri login
  • Data Santri
    Detail data pribadi santri
  • Pembayaran Santri
    Informasi dan manajemen pembayaran
  • Ujian/Tes Masuk
    Informasi ujian dan hasil tes
  • Cetak Dokumen
    Halaman untuk mencetak berbagai dokumen

Halaman Admin

  • Dashboard Admin
    Halaman utama panel admin
  • Kelola Santri
    Daftar dan manajemen data santri
  • Detail Santri
    Halaman detail data santri
  • Pembayaran Admin
    Manajemen pembayaran santri
  • Detail Pembayaran
    Detail dan verifikasi pembayaran
  • Kelola Gelombang
    Manajemen gelombang pendaftaran
  • Form Gelombang
    Form tambah/edit gelombang
  • WhatsApp Config
    Konfigurasi WhatsApp API
  • WhatsApp Template
    Manajemen template pesan WhatsApp
  • WhatsApp Admin
    Manajemen admin penerima notifikasi
  • WhatsApp Broadcast
    Pengiriman pesan broadcast
